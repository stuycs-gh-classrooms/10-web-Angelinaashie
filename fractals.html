My altered tree has certain modifications that make it distinct from the original tree: it carries the colors brown and green, much like an actual tree does, and this color distribution changes by depth. Also, the angle measure of the turtle is changed from 20 to 30, giving the altered tree its differing shape. 


      def tree(t, depth, size, angle):

A Special Snowflake

The original koch curve is of a uniform color and depth, but the modifications made to the code using the randrange function allows for randomization of color and depth, meaning each time the code is run the koch curve appears as a different color and a different snowflake variation, with a depth between 1 and 5. 


koch_turtle = turtle.Turtle()
x = random.randrange(0, 255)
y = random.randrange(0, 255)
z = random.randrange(0, 255)
koch_turtle.color(x,y,z)
koch_turtle.pensize(3)
koch_turtle.penup()
koch_turtle.setpos(-100, 0)
koch_turtle.pendown()
koch_turtle.speed(0)

def triangle(size):
    for angle in [-120, -120, 0]:
        koch(koch_turtle, depth, size)
        koch_turtle.left(angle)

def koch(koch_turtle, depth, size):
    if depth == 0:
        koch_turtle.forward(size)
    else:
        for angle in [60, -120, 60, 0]:
            koch(koch_turtle, depth - 1, size // 3)
            koch_turtle.left(angle)
size = 200
depth = random.randrange(1, 5)
triangle(size)
